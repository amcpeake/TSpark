LEGO COMMANDS:
!define [word] ~ Defines a given word
	* -n [#] ~ Returns a given number of definitions

!pyde ~ Run code and return the output (STDOUT)
	* -l [language] ~ Specify language of code
	* -m [msgID] ~ Runs given message as code. Must use \`\`\`[language][code]\`\`\` format
	* -c [code] ~ Runs given string as code
	* -i [json] ~ Uses given JSON as input to the code as STDIN

!joke ~ Tells a joke

!regedit <registry> <value> ~ Display and edit registries (stored globals)
	* -l ~ List all registries
	* -l [registry] ~ Displays value of given registry
	* -a [registry] [value] ~ Add to a list registry

!roll ~ Gives you your message ID as a \"roll\"

!nab [emoji] ... ~ Grab messages contained between last two instances of messages with given reactions'
	* -c [#channel] ~ Search in a given channel (ctx.channel)
	* -n [all|num] ~ Search a given number of previous messages (1000)

!search [string] ~ Search for messages
	* -c [all|#channel] ~ Set channel to search through (ctx.channel)
	* -n [all|num] ~ Set number of messages to read (1000)
	* -u [all|@user] ~ Set the user to read messages from (ctx.author)
	* -r [emoji] ~ Set the required reaction to be read

!moji [name] ~ Sends a moji
	* -l ~ List available mojis
	* -a [name] [link] ~ Add a moji
	* -r [name] ~ Remove a moji

!reminder [reminder_text] [days|hours|minutes] ~ Set a reminder
	* -u [@user] ~ Set a reminder for a user (ctx.author)
	* -l ~ List all reminders

!discloud ~ Store and retrieve files
	* -l ~ List all stored files and their indices
	* -g [index] ~ Get a file given its index
	* -s [messageID] ~ Store a file from a message id

WAK COMMANDS:
!eval ~ Evaluates a given string

!img ~ Searches google for string and returns first image

!wiki ~ Searches wikipedia for string and returns first result

!play [game] ~ Adds a game for Tony to play

!history ~ Dumps all messages from a channel
